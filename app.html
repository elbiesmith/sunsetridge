Sunset Ridge: A coding challenge.

An array of mountains is facing the sun. The heights of each mountain from West to East is given in an integer array. You have to tell which mountain will be able to see the sunset. Write a program that prints how many mountains and which mountains will be able to see the Sunset given the Array [3, 2, 4, 7, 6, 9].

Instructions

If you just want to use the default array, click the Check Array Button. For a custom array of ANY length, click use custom array, and input your numbers into the input box. Then either click Add or press enter to add your number to the array.

If you make a mistake, click Undo to undo the last number entered or Reset to clear the whole array.
